1. Visión General
Proyecto: Framework de Gestión Empresarial FoodMaster.

Objetivo: Proporcionar un sistema de gestión de alto rendimiento, escalable y ágil, adaptado para entornos de cocina (mesas/comandas) y domicilios (pedidos).

2. Funcionalidades Core (MVP)
Gestión de Comandas: Control de estados de mesa para consumo presencial.

Gestión de Pedidos: Flujo completo para domicilios (recepción, preparación, despacho).

Persistencia: Almacenamiento seguro, transaccional y consistente mediante PostgreSQL.

Conectividad: Sincronización en tiempo real entre el Frontend (Angular) y el Backend (Python/Flask).
